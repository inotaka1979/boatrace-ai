## 0. ゴールと原則

### 0.1 ゴール
1. **観測中の障害根絶**: heartbeat 誤検知アラート、git rebase 衝突、odds 二重起動を停止
2. **セキュリティ事故ゼロ化**: GitHub PAT の漏洩経路全閉塞、XSS 経路の完全エスケープ
3. **AI 予想の数値健全性**: NaN/Infinity 伝播、ゼロ除算、日付ずれの撲滅
4. **PWA としての成立**: Service Worker 登録の復活、iOS 7日問題の緩和
5. **保守容易性**: 死コード除去、設計の単一化（cron vs systemd の重複解消）

### 0.2 設計原則
- **後方互換**: localStorage キーのスキーマは破壊変更しない（マイグレーション関数で吸収）
- **アトミック性**: 全ての永続化（JSON / heartbeat / settings）は temp + rename
- **fail-loud**: silent fail を禁止。失敗は必ず log と非ゼロ終了で外に出す
- **単一責任**: 各 cron ジョブは 1 種類のデータのみ触る、重複スクリプトは削除
- **段階リリース**: Phase 0 → 5 の順で投入、各 Phase 後に観測 24h

### 0.3 非対象
- AI 予想アルゴリズムの仕様変更（重み学習則・特徴量設計）は扱わない
- UI の見た目変更（色・レイアウト）は扱わない
- 新機能追加（プッシュ通知・OAuth など）は扱わない

```bash
# scripts/cron_scrape.sh
set -euo pipefail
LOG_DIR="/home/pi/boatrace-ai/logs"
mkdir -p "$LOG_DIR"

update_heartbeat() {
  local mode="$1"
  local hb="${LOG_DIR}/heartbeat_${mode}.txt"
  local tmp
  tmp="$(mktemp "${hb}.XXXX")"
  if ! date '+%Y-%m-%d %H:%M:%S' > "$tmp"; then
    log "ERROR: heartbeat tmp write failed for ${mode}"
    return 1
  fi
  if ! mv -f "$tmp" "$hb"; then
    log "ERROR: heartbeat rename failed for ${mode}"
    rm -f "$tmp"
    return 1
  fi
  log "heartbeat updated: ${mode}"
}

run_scrape() {
  local script="$1" label="$2"
  log "Running ${label}..."
  local code=0
  "$PYTHON" "scripts/${script}" >> "$LOG_FILE" 2>&1 || code=$?
  if [ "$code" -ne 0 ]; then
    log "ERROR: ${label} failed (exit=${code})"
    return "$code"
  fi
  log "${label} completed"
  update_heartbeat "$label" || true   # heartbeat 失敗は単独で警告、本処理は成功扱い
}
```

```bash
check_heartbeat() {

  if [ ! -f "$hb" ]; then
    alert "${mode} heartbeat file missing"
    return
  fi
  local mtime

rotate_alert_log() {
```

```python
# scripts/_io.py（新設）
import json, os, tempfile
def atomic_write_json(path: str, data, *, encoding='utf-8'):
    d = os.path.dirname(os.path.abspath(path)) or '.'
    fd, tmp = tempfile.mkstemp(prefix='.tmp_', dir=d)
    try:
        with os.fdopen(fd, 'w', encoding=encoding) as f:
            json.dump(data, f, ensure_ascii=False, separators=(',', ':'))
            f.flush()
            os.fsync(f.fileno())
        os.replace(tmp, path)
    except Exception:
        try: os.unlink(tmp)
        except OSError: pass
        raise
```

#### L-06/L-10: softmax 安全化
```javascript
function softmax(logits){
  if (!Array.isArray(logits) || logits.length === 0) return [];
  var clean = logits.map(function(v){ return Number.isFinite(v) ? v : 0; });
  var max = Math.max.apply(null, clean);
  if (!Number.isFinite(max)) max = 0;
  var exps = clean.map(function(v){ return Math.exp(Math.min(v - max, 50)); });
  var sum = exps.reduce(function(a,b){ return a+b; }, 0);
  if (sum === 0) return clean.map(function(){ return 1/clean.length; });
  return exps.map(function(x){ return x/sum; });
}
```

#### W-01/W-02/W-03: sw.js リファクタ
```javascript
// sw.js v4
const VERSION = 'br-oracle-v4';
const STATIC = ['./manifest.json','./icon-192.png','./icon-512.png'];

self.addEventListener('install', e => {
  e.waitUntil(caches.open(VERSION).then(c => c.addAll(STATIC)));
  // skipWaiting は activate 後の判断に委ねる（自動更新通知ベース）
});

self.addEventListener('activate', e => {
  e.waitUntil((async () => {
    const keys = await caches.keys();
    await Promise.all(keys.filter(k => k !== VERSION).map(k => caches.delete(k)));
    await self.clients.claim();
  })());
});

self.addEventListener('message', e => {
  if (e.data === 'SKIP_WAITING') self.skipWaiting();
});

async function netFirst(req){
  try {
    const resp = await fetch(req);
    if (resp.ok) {
      const cache = await caches.open(VERSION);
      await cache.put(req, resp.clone());
    }
    return resp;
  } catch(_) {
    const cached = await caches.match(req);
    if (cached) return cached;
    return new Response(JSON.stringify({error:'offline'}), {
      status: 503, headers: {'Content-Type':'application/json'}
    });
  }
}

self.addEventListener('fetch', e => {
  const url = new URL(e.request.url);
  if (url.pathname.endsWith('/') || url.pathname.endsWith('/index.html')) {
    e.respondWith(netFirst(e.request)); return;
  }
  if (url.pathname.includes('/data/') || url.host === 'boatraceopenapi.github.io') {
    e.respondWith(netFirst(e.request)); return;
  }
  e.respondWith(caches.match(e.request).then(c => c || fetch(e.request)));
});
```

#### W-10: stale-while-revalidate
fetchWithFallback を以下に置換:
```javascript
async function swr(url, key, ttlMs){
  var cached = safeParse(key, null);
  var fresh = (async () => {
    var r = await fetch(url, {cache:'no-cache'});
    if (!r.ok) throw new Error('http '+r.status);
    var j = await r.json();
    safeSet(key, {time:Date.now(), data:j});
    return j;
  })().catch(e => { console.warn('[swr] fetch fail', url, e); return null; });

  if (cached && Date.now() - cached.time < ttlMs) return cached.data;
  var got = await fresh;
  return got || (cached && cached.data) || null;
}
```

```ini
# /etc/systemd/system/boatrace-scrape@.timer
[Unit]
Description=BoatRace scrape timer (%i)
[Timer]
OnCalendar=*:0/3
Unit=boatrace-scrape@%i.service
[Install]
```
